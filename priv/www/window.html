Usage

Session 1: How to create a window

As you'll see, given that you have a basic understanding of XML or HTML markup, things are so trivial that barely any comment is necessary.

To create a window, create a file called tutorial.xml and type in the following...

<?xml version="1.0"?>
<gx>
  <window/>
</gx>

To see the result, open an Erlang shell and enter gx_runner:start("tutorial.xml") to see the results of your hard work.

1>gx_runner:start("tutorial.xml").
<0.262.0>

Next let's give this window a title...

<?xml version="1.0"?>
<gx>
  <window title="GX Tutorial"/>
</gx>
